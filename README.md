# EventPulse
منصة إيرادات ذكية مدعومة بالذكاء الاصطناعي تُحسّن ديناميكياً تسعير الفنادق في السوق السعودي وفقاً لتحليل الأحداث الحية والتنبؤ بالطلب، باستخدام خرائط حرارية ونماذج تنبؤية لزيادة الإيرادات بنسبة تصل إلى 45% خلال الفترات عالية الطلب (مثل رمضان والفعاليات الكبرى)، مع توفير رؤى استراتيجية فورية لاتخاذ قرارات مُدروسة تُعزز الربحية وتقلل المخاطر.
