# مشروع SmartHotel AI - نظام التسعير الاستراتيجي

## نظرة عامة

هذا المشروع يتكون من نظامين رئيسيين:
1. **واجهة API** لجلب وتحديث بيانات الفعاليات السعودية
2. **لوحة تحكم ذكية للفنادق (SmartHotel AI)** لنظام التسعير الاستراتيجي 2026

## الملفات المكونة للمشروع

### 1. api.html
واجهة المستخدم لتحديد موقع الفعاليات ونوع التحديث وتحديث البيانات من خلال Google Apps Script.

#### الميزات:
- اختيار المحافظة (Riyadh, Jeddah, Dammam, Makkah, Madinah, etc.)
- اختيار نوع التحديث (Google Calendar, فعاليات السعودية, مواسم السعودية, أخبار محلية)
- عرض بيانات الشيت (Sheet Data) في جدول قابل للتحديث
- واجهة باللغة العربية

#### التكنولوجيا المستخدمة:
- HTML5
- CSS3
- JavaScript
- Google Apps Script (API)

### 2. dashboard.html
لوحة تحكم ذكية للفنادق لتحليل بيانات الفندق وتقديم توصيات تسعير استراتيجية.

#### الميزات:
- إدخال بيانات الفندق الأساسية (اسم الفندق، المنطقة، عدد الغرف، السعر الأساسي)
- إدارة أنواع الغرف وعدد كل نوع
- تحليل الإيرادات المتوقعة
- عرض تأثير الأحداث على الإشغال
- عرض الخرائط الحرارية للأسعار والإشغال
- إعدادات استراتيجية التسعير الذكية

#### الاستراتيجيات:
- **الحجز المبكر**: زيادة الأسعار في رمضان/الحج بنسبة 35-45%
- **التسعير الديناميكي**: تعديل يومي مع أحداث خاصة (يوم التأسيس/اليوم الوطني)
- **تسعير الأحداث**: زيادة 40% للأحداث الكبرى (فورمولا/مؤتمرات)
- **حزم الترقية**: إضافة وجبات/نقل (+6% ADR)
- **حزم الشركات**: عروض المؤتمرات والمعارض (+20%)
- **الحد الأدنى للإقامة**: فرض ليلتين في الذروة

### 3. ملفات CSS

#### css/events.css
- تنسيقات واجهة API
- ألوان وأشكال لعناصر التحكم
- جداول عرض البيانات
- مؤشرات التحميل

#### css/dashboard.css
- تنسيقات لوحة التحكم الذكية
- بطاقة شفافة (Glass Panel)
- رسم بياني وخرائط حرارية
- إعدادات الاستراتيجيات
- استجابة الشاشات المختلفة (Responsive Design)

### 4. ملفات JavaScript

#### js/events.js
- الاتصال بـ Google Apps Script
- إرسال طلبات التحديث
- جلب وعرض بيانات الجدول
- وظائف التحديث الفوري

#### js/dashboard.js
- تحليل بيانات الفندق
- حساب الإيرادات والتكاليف
- تطبيق استراتيجيات التسعير
- الاتصال بـ Google Apps Script للحصول على معلومات الأحداث
- عرض النتائج بطريقة مرئية

## المكونات التفاعلية

### API Interface
- تحديد المحافظة من قائمة منسدلة
- تحديد نوع التحديث باستخدام أزرار الراديو
- زر تحديث البيانات
- عرض نتيجة العملية (نجاح أو خطأ)
- جدول عرض البيانات مع إمكانية التعديل

### Dashboard Interface
- نموذج إدخال بيانات الفندق
- عرض ملخص الغرف
- عرض الإحصائيات (الإيراد السنوي، صافي الربح، هامش الربح، معدل الإشغال)
- عرض الخرائط الحرارية الشهرية
- لوحة إعدادات الاستراتيجيات

## التكامل مع Google Apps Script

- يتم استخدام Google Apps Script كطبقة ويب لدمج البيانات
- الربط مع Google Sheets لتخزين وتحديث البيانات
- جلب معلومات الأحداث تلقائيًا
- مزامنة البيانات بين النظام والخوادم

## ميزات الذكاء الاصطناعي

- التسعير التنبؤي استنادًا إلى الأحداث
- اقتراحات التسعير المثلى
- تحليل الأنماط الموسمية
- اقتراحات لتحسين الهامش الربحي

## ملاحظات تقنية

- المشروع مصمم ليكون باللغة العربية
- يدعم الاتجاه من اليمين إلى اليسار (RTL)
- متوافق مع مختلف أحجام الشاشات
- يعتمد على Chart.js لعرض الرسوم البيانية
- يستخدم SheetJS لتصدير ملفات Excel

## الاستخدام

1. قم بتشغيل `api.html` لعرض واجهة API
2. قم بتشغيل `dashboard.html` لعرض لوحة التحكم
3. قم بتعيين روابط Google Apps Script في ملفات JavaScript
4. استخدم الواجهات لجلب وتحديث البيانات وتحليل أداء الفندق

- متصفح ويب حديث
