# دليل المساهمة

مرحبًا بك في مشروع SmartHotel AI - نظام الذكاء الاصطناعي لتسعير الفنادق! نحن سعداء لرغبتك في المساهمة في هذا المشروع المفتوح المصدر.

## كيف يمكنك المساهمة

### الإبلاغ عن الأخطاء
إذا وجدت خطأ برمجيًا أو مشكلة في النظام، يرجى فتح "Issue" جديدة مع وصف مفصل للمشكلة وخطوات إعادة إنتاجها.

### طلبات الميزات
نرحب بأفكارك ومقترحاتك لتحسين النظام. يرجى فتح "Issue" لمناقشة الميزات الجديدة قبل تنفيذها.

### إرسال تعديلات الكود
1. **افصل (Fork) المشروع** على حسابك الشخصي
2. **أنشئ فرعًا جديدًا** (`git checkout -b feature/NewFeature`)
3. **قم بتعديل الكود** وفقًا لمتطلباتك
4. **-commit التغييرات** (`git commit -m 'Add some NewFeature'`)
5. **ادفع التعديلات** (`git push origin feature/NewFeature`)
6. **أرسل Pull Request**

## إرشادات المساهمة

### معايير الكود
- اتبع معايير JavaScript الحديثة
- استخدم تعليقات واضحة ومفيدة
- تأكد من توافق الكود مع المتصفحات الحديثة
- صِم الكود ليكون مناسبًا للبيئة العربية

### نمط الكتابة
- استخدم اللغة العربية بشكل سليم
- حافظ على تنسيق الكود مرتبًا
- استخدم أسماء متغيرات واضحة وقابلة للفهم

### اختبارات الكود
- قم باختبار الكود الخاص بك بشكل كامل
- تأكد من أن الكود لا يحتوي على أخطاء برمجية
- اختبر الكود في متصفحات مختلفة

## بيئة التطوير

### المتطلبات
- متصفح ويب حديث (Chrome، Firefox، Safari)
- محرر نصوص (VS Code موصى به)

### الإعداد
1. Clone المشروع
2. فتح الملفات محليًا
3. تشغيل الملف الرئيسي `index.html`
